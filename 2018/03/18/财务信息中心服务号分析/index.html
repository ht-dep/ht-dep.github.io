财务信息中心服务号分析

发表于 2018-03-18 | 分类于 公众号 | | 阅读次数：

从自身的需求出发，开发了本服务号
方便员工获取、跟踪信息
提供统一的记录地点

功能介绍

  1. 员工信息
  2. 工资查询
  3. 报销跟踪
  4. 智能助手
  5. 公司信息（比如发票）

员工信息

  1. 第一次关注时，需要填写员工信息并提交
  2. 查看员工个人信息
  3. 如果修改信息，需要通知管理员，将状态修改为可更新

工资查询

查询类别

  1. 月收入（工资）
  2. 项目可（津贴）
  3. 年度统计

查询方式

  1. 点击推送
  2. UI交互
  3. 输入交互

三种查询方式，可以更大程度上满足用户的查询需求，并更大程度上保障了在网络不好时的一个查询结果，比如：输入交互通常比UI交互快一些

发票信息

大家在群里沟通时，提到发票信息不能以一种友好的方式进行查询、保存

点击推送，图文格式的发票信息—美观

  1. 一张图片，无法编辑

输入”发票”，文本格式的发票信息—不美观

  1. 可以进行复制、编辑
  2. 迎合不同宾馆填写发票多种方式的需求，比如：有的宾馆需要用户通过微信去填写

报销跟踪

工作生活中，发现不能很好地跟踪报销的进度、核实对比报销金额

创建报销

  1. 选择不同的出差类别
  2. 跟出差类别，自动计算出差补贴

报销记录

  1. 显示已填写未处理的所有报销
  2. 工资卡收到到款提醒时，比对金额无误时，点击处理，该报销记录完成（该记录消失）

报销跟踪，需要员工个人进行维护

智能助手

提供一些辅助功能

  • 天气、翻译、计算器、菜谱
  • 百科、故事、笑话、运势
